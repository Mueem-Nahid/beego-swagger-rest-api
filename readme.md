## Project: Beego, Swagger, REST API
**Requirments:**

 - You will need a [Go](https://golang.org/) 1.1+ installation for this to work.
 - You will need to install or upgrade [Beego](http://beego.vip/docs/install/bee.md) and the [Bee](http://beego.vip/docs/install/bee.md) dev tool:
 - ````go get -u github.com/beego/beego/v2````
 - ````go get -u github.com/beego/bee/v2````
 - PostgreSQL needed or PGadmin 4 needed
 - Create a database `user_db`, table will be created automatically
 - Download or clone the repository
 - Go to the downloaded folder and open the project folder
 - Run `go mod tidy`
 - And run `bee run`
 - You are all set. Now go to the beowser and hit`http://localhost:8080/swagger/`, you will see like this: ![enter image description here](https://i.ibb.co/5RZ2yDN/Untitled.png)
 - From swagger, you can test the api.

**About this project:**

 - Beego used
 - Beego form validation used
 - Postgresql used to store data
 - And many more
